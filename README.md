## Available flags

* `@web/root` - Acceso completo
* `@web/access` - Acceso a la pagina administrativa

* `@web/admin.create` - Permission to create an admin.
* `@web/admin.edit` - Permission to edit an admin.
* `@web/admin.delete` - Permission to delete an admin.

* `@web/ban.view` - Ver lista de bans
* `@web/ban.add` - Añadir un ban
* `@web/ban.edit` - Editar un baneo
* `@web/ban.unban` - Desbanear un baneo
* `@web/ban.remove` - Remover un baneo

* `@web/mute.view` - Ver la lista de muteos
* `@web/mute.add` - Añadir un mute
* `@web/mute.edit` - Editar un muteo
* `@web/mute.unmute` - Desbanear un muteo
* `@web/mute.remove` - Remover un muteo

* `@web/group.create` - Permission to create a group.
* `@web/group.edit` - Permission to edit a group.
* `@web/group.delete` - Permission to delete a group.

* `@web/search.players` - Posibilidad de buscar jugadores (mediante los logs de conexiones)
